Profile Tab

The Profile Tab in Venue Driver is where a user can view and edit their own information.

Each field is pretty self-explanatory, but will be detailed below for reference.

First Name
The firts name of the user.

Last Name
The last name of the user.

Email
The email of the user.

Password: (or blank to leave unchanged)
The password of the user, you can leave this field blank to leave unchanged

Password: (again)
If you change your password please repeat it.
