Entering returns into Venue Driver

Venue Driver allows for comprehensive tracking of VIP customer spends and guestlist returns. This data is then passed onto the Staff Performance section and also the Customers tab. Guestlist returns can either be entered automatically via the iOS app or web interface check-in, or manually in this returns section. VIP customer spends can be entered manually during or after an event. Automated VIP spend tracking via an add-on POS interface will be deploying in the latter half of 2011.

To access the returns for a given date, select the date using the calendar icon in the upper left-hand corner of the browser window.


Entering VIP Reservation returns


To enter VIP reservation returns, click on the button marked 'Reservations' in the event listing for that day. A list of the event's reservations will then present itself. The fields can either be edited inline by hovering over the field and clicking on them to edit (a 'Click To Edit' tooltip will appear), or they can be edited by clicking the 'Edit' button.

The fields, from left to right, are as follows:

First
The first name of the client. This is not editable.

Last
The last name of the client. This is not editable.

Actual Guests
The actual number of guests who were sat at a table. In order for a reservation to be credited to a staff member and tracked, a number must be entered here.

Actual Paid Bottles
The number of paid bottles for this reservation.

Actual Comp Bottles
The number of comp (free) bottles for this reservation.

Actual Spend
The actual amount of money spent by this client.

Staff
The staff member who booked the reservation. This cannot be edited.

Area
The area the reservation was booked in.

Table
The specific table the reservation was booked in.
